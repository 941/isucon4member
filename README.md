- チーム代表かチームメンバー希望か

- 使用言語
- 担当分野
- 予選参加日（27,28,どちらでも）

idobataの「ISUCON4メンバー募集」に入り相談してチームメンバーを決めてください。
https://idobata.io/#/organization/isucon4member
